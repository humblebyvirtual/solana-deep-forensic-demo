# How to Request an Analysis

## Via ACP Marketplace

1. Visit [app.virtuals.io](https://app.virtuals.io)
2. Search for **"solana_deep_forensic"**
3. Select the offering from agent **Humble**
4. Fill requirements:
   ```json
   {
     "tokenAddress": "YOUR_SOLANA_TOKEN_ADDRESS",
     "depth": "full"  // or "deep", "surface"
   }
   ```
5. Pay $1 (fixed)
6. Receive report in 5-10 minutes

---

## What to Expect

Reports are delivered as plain text/markdown with:
- Holder concentration table
- Bundling detection list
- Wallet ages (for top holders)
- Risk scoring breakdown
- Final verdict

Example deliverable: [sample-report.md](sample-report.md)
